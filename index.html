Esta guía tiene como objetivo proporcionar información sobre cómo proteger la BIOS/UEFI y habilitar el arranque seguro en Windows. También discutiremos las contraseñas y la seguridad del sistema operativo. Esta guía es útil tanto para principiantes como para aquellos con experiencia limitada en seguridad de sistemas.

La BIOS (también conocida como UEFI) es un componente esencial de cualquier sistema operativo porque controla el arranque y la configuración del hardware. En esta sección, hablaremos sobre cómo proteger la BIOS/UEFI y garantizar que los ajustes de sistema críticos estén seguros. Aprenderás cómo establecer contraseñas, evitar restablecimientos no autorizados y mejorar la seguridad de esta parte importante de tu computadora.

  1. Reinicia tu computadora y presta atención a la pantalla durante el inicio. Por lo general, muestra una tecla para ingresar a la BIOS/UEFI (por ejemplo, F2, F12, Del).
  2. Ingresa a la BIOS/UEFI siguiendo las instrucciones.
  • Configura una contraseña de administrador de la BIOS/UEFI para evitar cambios no autorizados.
  • Configura una contraseña de usuario de la BIOS/UEFI para proteger los ajustes básicos.

Si deseas una capa adicional de seguridad, configura una contraseña de arranque del dispositivo. Esto evitará que se inicie el sistema sin la contraseña.
  • Ten en cuenta que algunos métodos de reinicio pueden anular las contraseñas de la BIOS/UEFI.
  • Investiga si tu placa base tiene una opción para evitar el restablecimiento de contraseña.

El componente crucial del arranque seguro de Windows permite que el sistema no ejecute software no autorizado durante el proceso de inicio. En esta sección, aprenderás sobre el mundo del arranque seguro, que incluye el uso del módulo de plataforma confiable (TPM) y Secure Boot. Instrucciones sobre cómo verificar y habilitar estas características de Windows, que son esenciales para mantener la integridad del sistema operativo, se proporcionarán en este curso.

  1. Abre Administrador del módulo de plataforma segura (tpm.msc) en tu sistema Windows.
  2. Verifica si tu computadora tiene activado el TPM (Compatibilidad con cifrado de dispositivo) y Secure Boot (Estado de arranque seguro).
  • Si Secure Boot no está habilitado, accede a la BIOS/UEFI siguiendo el Paso 1 de la Sección 1.
  • Busca la opción Secure Boot y actívala. Esto ayudará a prevenir la ejecución de software no firmado en el arranque.
  1. Configura una contraseña sólida para tu cuenta de Windows. No uses contraseñas obvias o débiles.
  2. Evita usar preguntas de seguridad débiles para la recuperación de contraseñas.

Si deseas mejorar la seguridad, investiga cómo deshabilitar o restringir la recuperación de contraseñas en tu versión de Windows. Esto puede variar según la edición de Windows que estés utilizando.

Esta guía proporciona los pasos necesarios para mejorar la seguridad de su BIOS UEFI y habilitar el arranque seguro de Windows.Además, enfatiza la importancia de configurar contraseñas fuertes y considerar deshabilitar la recuperación de contraseñas débiles para una mayor seguridad.

Recuerda que la seguridad nunca es 100% perfecta, tomar medidas como estas puede ayudarlo a proteger su sistema contra accesos no autorizados.
